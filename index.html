General idea

This webpage is some version of notes from the 6 lecture series. Right now, they are very rough notes, texed live during the talk. Part of the point of sharing them on GitHub is that potentially other people at the conference can help clean them up and flesh them out. Depending on how ambitious we get, we could make some very nice notes or even a book: for instance, the Homotopy Type Theory
was written by a large group of mathematicians using GitHub; their GitHub repository is here, and you can read something about the process by which it was written here.

The HoTT book is much more ambitious than I have in mind now, which was more along the lines of "can I TeX notes from a talk live, and can I use GitHub", but we'll see what happens.

Git has a reputation of being a bit complicated, but there are a large number of tutorials and help things on the web; I'm teaching myself as we go along.
